# Contributing

Thanks for helping improve Audio Levels.

## Development setup

- Python 3.9+
- `ffmpeg` available in your `PATH`

```sh
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt
```

Run locally:

```sh
.venv/bin/python serve.py
```

## Making changes

- Keep changes focused and small when possible.
- Prefer clear, descriptive names over abbreviations.
- If you change behavior or flags, update `README.md`.

## Submitting a PR

- Fork the repo and create a topic branch.
- Open a PR with a clear description and screenshots/screen recordings for UI changes when applicable.
